# Wallets

Wallets listed below support the Gnosis Chain.&#x20;

For GNO on Ethereum, you can use any [Ethereum based wallet](https://ethereum.org/en/wallets/) and add the GNO token ([0x6810e776880C02933D47DB1b9fc05908e5386b96](https://etherscan.io/token/0x6810e776880C02933D47DB1b9fc05908e5386b96)) to your tokens list.
